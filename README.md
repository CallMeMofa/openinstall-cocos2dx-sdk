使用 `JavaScript` 集成 openinstall SDK 到 cocos2d-x 项目中，请使用 `JavaScript` 目录的文件，并参考 [JavaScript 集成指南](JavaScript/README.md)

## Lua 集成
使用 `Lua` 集成 openinstall SDK 到 cocos2d-x 项目中，请使用 `Lua` 目录的文件，并参考 [Lua 集成指南](Lua/README.md)
